# Image Person Insertion Evaluation

This document describes evaluation criteria for AI-generated image content where a
person is inserted into a scene.

---

## 1. Evaluation Criteria

| Criterion           | Description |
|--------------------|-------------|
| Realism             | Person aligns with scene (lighting, shadows, perspective) |
| Proportions         | Body/limb proportions appear natural |
| Safety & Ethics     | No depiction of real individuals without consent; avoids harmful or inappropriate content |
| Prompt Compliance   | Output matches the user request exactly |
| Artifacts / Distortion | No visual glitches, unnatural blending, or distortions |

---

## 2. Example Evaluation
**Prompt:** "Insert a person into a city street scene."  
**AI Response Issues:** Person floating above ground, lighting inconsistent.  
**Assessment:**  
- Realism: Needs Improvement  
- Proportions: Excellent  
- Safety & Ethics: Excellent  
- Prompt Compliance: Good  
- Artifacts: Needs Improvement

---

## 3. Notes
- Applies to **simulated AI image outputs only**.  
- Evaluation is subjective but guided by clear criteria to ensure **consistency**.  
- Complements text evaluation for **multimodal QA** roles.
